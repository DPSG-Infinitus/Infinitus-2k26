Implementation Plan - Site Revisions & Asset Upgrades
This plan details the changes to remove the Gallery, add the newly uploaded logo.png logo, make the timeline interactive, adjust the event card photo dimensions, update contact email references, and embed a live Google Map for school navigation.

User Review Required
IMPORTANT

Key Refinements:

Gallery Removal:
The Gallery section and all navigation links pointing to #gallery (header and footer) will be deleted.
Timeline Interaction:
We will replace the static timeline grid with an interactive scheduling deck.
Users can click on date tabs (01st July, 15th July, 16th-18th July, 21st July) to dynamically toggle detail summaries below them.
The July 21st (Grand Finale) tab will present a detailed itinerary timeline grid (Reporting, Inauguration, Runs, Prize Distribution).
Navbar Logo:
The text crest "I" in the navbar and footer will be replaced with an image element loading the newly uploaded logo.png (assets/logo.png).
Event Photo Dimensions:
The left-side poster box on each event card will be set to a fixed, clean width of 180px (with column stacking on mobile), preventing excessive stretching or cropping.
Live Campus Map & Contact Email:
Replace the map placeholder with a live, responsive iframe embedding Delhi Public School Ghaziabad, Meerut Road from Google Maps.
Update all support email references to infinitus2026@gmail.com.
